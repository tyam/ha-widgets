ラジオボタン


API
--------

### 継承

[Component](component)

### パラメータ

| 名前 | 型 | デフォルト値 | 説明 |
| ---- | -- | ----------- | ---- |
| id | string | - | DOM要素のidを指定してください。 |
| invalid | boolean | `false` | 使用不可能性 |

その他、htmlのinput(type="radio")要素の属性を与えられます。

### 子要素

マークアップ。  
ラジオボタンのラベルとして表示されます。
